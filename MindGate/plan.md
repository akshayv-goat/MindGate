# MindGate — Initial Build Plan

## MVP focus

Start with the core problem and simplest working intervention:
1. User chooses an app to monitor.
2. User selects a dedication level: Light, Medium, Hardcore.
3. The app triggers a context-aware intervention when the selected app is used.
4. The app records whether the user continued or paused, and logs a reflection.

## Suggested first deliverables

- Product flow diagrams for:
  - onboarding + app selection
  - session intervention
  - progress dashboard
- Wireframes for:
  - game overlay
  - full-screen scroll pause
  - custom app settings
- A basic prototype using a web/mobile framework or design tool.

## Discovery questions

- Which apps should be supported first? (e.g. Roblox, YouTube, Instagram)
- What platform(s) are highest priority? (Android, iOS, web)
- How should parent summary be introduced without feeling invasive?
- What metrics matter most for early adoption?

## Recommended milestones

1. Define user personas and app contexts.
2. Validate the concept with a simple clickable prototype.
3. Build the MVP intervention flow and test with real teens.
4. Iterate on the emotional progress experience.
