# MindGate

MindGate is a context-aware accountability app designed to create a conscious moment of choice between a teenager and mindless screen consumption. It is built around the insight that different apps and usage modes require different interventions: games need lightweight overlays, scroll apps need intentional full-screen pauses, and each user should control their level of accountability.
